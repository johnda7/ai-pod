## Если не открывается в браузере:

1. **Проверьте что сервер запущен**:
   ```bash
   ps aux | grep vite
   ```

2. **Проверьте порт**:
   ```bash
   lsof -i :3000
   ```

3. **Откройте в браузере**: http://localhost:3000

4. **Откройте консоль (F12)** и проверьте ошибки

## Возможные проблемы:

- Браузер блокирует localhost
- Проблемы с CORS
- JavaScript ошибки в консоли
- Проблемы с загрузкой модулей из CDN

## Решение:

Пришлите скриншот или текст ошибок из консоли браузера (F12 → Console)
