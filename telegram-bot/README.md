# AI Pod Telegram Bot

Telegram бот для платформы AI Pod - геймифицированного обучения для подростков.

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

3. Заполните токен бота в `.env`:
```
TELEGRAM_BOT_TOKEN=ваш_токен_здесь
```

## Запуск

```bash
# Обычный режим
npm start

# Режим разработки (с автоперезагрузкой)
npm run dev
```

## Команды бота

- `/start` - Начать работу
- `/help` - Справка по командам
- `/profile` - Мой профиль
- `/tasks` - Список задач
- `/leaderboard` - Таблица лидеров
- `/katya` - Чат с AI-наставником
- `/week1`, `/week2`, `/week3` - Задачи по неделям

## Интеграция

Бот интегрирован с:
- Веб-приложением AI Pod
- Supabase (для хранения данных)
- Gemini API (для AI-функций)

## Разработка

Для расширения функциональности бота:
1. Добавьте новые команды в `index.js`
2. Интегрируйте с сервисами из папки `../services/`
3. Используйте типы из `../types.ts`
